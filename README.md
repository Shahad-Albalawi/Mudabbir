# Mudabbir | Graduation Project

Mudabbir is a cross-platform mobile application for personal finance management. It provides actionable insights to track expenses, monitor savings goals, and improve financial habits efficiently. The application transforms raw financial data into clear, structured guidance to support smarter financial decisions.

The system integrates intelligent analytics and an AI-powered chatbot to deliver personalized financial assistance in both Arabic and English.

---

## Overview

Mudabbir enables users to gain full visibility over their financial behavior through intuitive dashboards, behavioral analysis, and real-time insights. It combines tracking, analysis, and AI interaction into a unified experience.

---

## Key Features

- Expense tracking and categorization  
- Savings goal management and progress tracking  
- AI-powered chatbot for financial guidance (Arabic & English)  
- Behavioral analysis of spending patterns  
- Social challenges to encourage saving habits  
- Interactive dashboards for financial insights  

---

## Repository Structure

The project follows a clean monorepo layout:

- `apps/frontend` - Flutter mobile application  
- `apps/backend` - Laravel API (chatbot and challenges)  
- `docs` - Documentation and setup guides  
- `tools` - Development utilities and scripts
